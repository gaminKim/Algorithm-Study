# 알고리즘 스터디

현재 인원 : 5명

오프라인/ 온라인 격주 운영

## __참석관리__

2달에 패스권 3개 제공( 온라인 0.5 / 오프라인 1 ) => 2달동안 온라인 2번(0.5*2=1) 오프라인2회(1 * 2=2) 불참가능, 이후 불참에 대해서는 벌금 5천원 발생

오프라인 모임 시, 스터디룸비 발생 (5천원정도)
